## API

| 参数    | 说明                   | 类型                      | 默认值  |
| ------- | ---------------------- | ------------------------- | ------- |
| size    | 自定义 Icon 图标的大小 | `px`                      | `16`    |
| color   | 自定义 Icon 图标的颜色 | `string`                  | `block` |
| loadIng | 组件是否旋转           | `Boolean`                 | `false` |
| onClick | 点击图标回调           | `React.MouseEventHandler` | ——      |
